코더스란?

코더스는 풀스택 개발을 수월하게 만들어주는 도구로서, 초보자부터 전문 개발자까지 활용할 수 있는 통합 개발 환경을 제공합니다.

1. 풀스택 개발자의 어려움​

풀스택 개발은 서버, API, 데이터베이스, 화면 설계 등 다양한 분야의 지식이 요구됩니다. 코더스는 이러한 복잡성을 줄이고 개발 과정을 간소화해줍니다.

2. 화면(Frontend) 만들기​

2.1 Flutter 화면 생성 지원​

  • Dart 기반의 Flutter 개발을 JSSP 문법으로 변환하여 제공합니다.
  • 내장 서버로 테스트와 데모 버전 제작이 용이합니다.

2.2 Embed 데이터베이스와 Local Storage​

  • Embed 데이터베이스를 설계할 수 있는 JSSP 문법 제공.
  • Local Storage로 데이터를 간단하게 저장하고 관리할 수 있습니다.

3. 서버(Backend) 만들기​

3.1 외부 라이브러리 연동​

  • 다양한 외부 라이브러리와 연동 가능한 문법 제공.
  • Backend 서버 구성이 가능하며, 이미지, 파일 등을 저장할 수 있는 JSSP 문법 확장 계획.

3.2 문서화​

  • API 사양 문서화, 데이터베이스 문서화 등을 자동화합니다.

3.3 보안​

  • 설정 파일 암호화로 보안 강화.

4. 설치 및 지원​

4.1 Coders 설치​

  • Nuget.org에서 설치.
  • 명령어: dotnet tool install --global coders

4.2 JSSP Extensions 설치​

  • Visual Source Code에서 "jssp"로 검색하여 설치.
  • Syntax Highlight 기능 제공.

5. 향후 확장 계획​

  • 메모리 DB 연동, NoSQL 연동.
  • 이메일 전송, SMS 등 Backend에 필요한 클라우드 기술 확장.
  • Websocket API, RPC, TCP/IP 등의 프로토콜 설계 지원 계획.

6. 결론​

코더스는 Frontend와 Backend를 JSSP 하나의 문법으로 통합하여 작성할 수 있는 매우 강력한 도구입니다. 개발 과정의 복잡성을 줄이고, 효율성을 높여줄 것으로 예상됩니다.
